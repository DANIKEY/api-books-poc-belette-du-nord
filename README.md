## Description

Api books nodesjs.


## Installation

```bash
# global installation
$ apt install nodejs 
$ apt install npm
$ npm i -g @nestjs/cli
```


```bash
$ npm install
```

## config api books

```bash
open folder "src/"
edit file "app.module.ts"
config database username password
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
